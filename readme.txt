== Description ==

[ACLEEC](http://www.acleec.com/ "Nouveau systeme de micro-paiement") est un nouveau systeme de micro-paiement qui est specialement concu pour valoriser tres simplement
les contenus en ligne de petite valeur.

Lorsqu'un internaute a souscris un compte ACLEEC, il peut payer en cliquant simplement sur le montant demande.
De son cote le vendeur affiche dans son site le montant demande.

Le plugin ACLEEC pour WordPress gere entierement le processus de paiement. Les etapes de mise en oeuvre du paiement sont les suivantes :

1) **Installation du plugin sous WordPress**

L'installation est standard, le plugin est dans le dossier acleec-wp

Le plugin cree une option de menu "paiement" dans le menu "reglages" du tableau de bord.

2) **Parametrage du paiement**

Dans le menu paiement, il faut au minimum saisir son numero de vendeur ACLEEC. Ce numero est donne sur le site http://www.acleec.com
lorsqu'on cree un compte vendeur. L'ouverture est entierement gratuite.

D'autres parametrages sont possibles et auto-documentes dans la page de reglage.

3) **Choix des montants**

Pour chaque page payante, il suffit d'indiquer le montant demande. Soit la page est payante soit une contribution volontaire est proposee
en fin d'article.

*	Page payante : seul le resume est affiche. En bas du resume, un ticket de paiement est affiche. L'internaute clique sur le montant et
a acces a l'integralite de la page

*	Contribution : la page est affichee et en bas, l'internaute a la possibilite de donner on cliquant sur le ticket de paiement.

4) **Suivi des paiements**

Pour chaque page, l'auteur du blog peut visualiser le montant des ventes.


== Installation ==

1.	Copier les fichiers du plugin dans dans /wp-content/plugins/acleec-wp
1.	Activer le plugin dans WordPress
1.	Saisissez votre code vendeur dans la page "paiement" cree dans le menu "reglages" du tableau de bord
1.	Pour chaque page payante, saisissez le montant que vous en demandez

== Frequently Asked Questions ==

= Combien ca coute ? =

L'ouverture d'un compte acheteur ou d'un compte vendeur est completement gratuite.

= Quelle sont les devises acceptees ? =

A terme, ACLEEC acceptera la majorite des devises : euros, dollars, livrs, yens. La conversion est automatique.
Pour le moment, ACLEEC est en cours de developpement est elargit sa base d'utilisateurs actifs en acceptant
l'argent virtuel appelee "try money"
